About Game

Coding Quizee offers a wide variety of programming quizzes meant for knowledge that can serve as a kind of a learning aid. Repetition is an important element of learning, and one way to repeat content is through quizzes. Quizzes in learning can be used to motivate learners, help them remember what they learned, and assess what they learned. Evidence from research shows us this and can be used to our advantage, if we follow a few simple principles mentioned above to make our quizzes more effective. So, if you are new to programming, then this a right place to begin with. Quizzes will be updated timely, as you go along you become more experienced, more proficient and more self-confident, and the best thing is that you get all that FOR FREE :-)! All that you have to invest is your time!

Question Page

Coding Quizee contains questions related to five programming languages(C, C++, C#, Java and Python). For each programming language, there are three levels(Basic, Intermediate and Advanced). In Question page, the top sections contains progress bar and submit button, then question and at last options. Questions are contained in carosuel, so users can go to any questions using arrows on sides.

Answer Page

Coding Quizee contains answers with explanation for all quizzes. Answer page of particular quiz can be accessed only after you attempt that. Top section of answer page contains title and back button to revert back to last page, then it conatins question, option, answer and explanation can seen after clicking view explanation button. Scrollbar on left takes you to desired question. It has answers to all programming languages.

Scorecard Page: Your Score

After attempting quiz, you can check scores in Your Score table of Scorecard Page. It contains date of attempted quiz, quiz type(like on which langauge and which level) and score of attempted quiz. Information will be sorted based on date.

Scorecard Page: Leaderboard

Leaderboard helps you in competing with others and it provides a way for you to improve over time. Table conatins rank, name and score for each quiz type. Above the table thers's a filter to filter data based on quiz type. Data is sorted for each quiz type.
